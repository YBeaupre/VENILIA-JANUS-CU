Steps required to run the JANUS demo without Venilia:

1. In the build directory, enter the JANUS data n the VeniliaDataBlock.txt file
2. Run MyJANUSDemo.mlx

Steps required to run the JANUS demo with Venilia:
1. Input relevant data in the Venilia Encode.mlx file
2. Run the Venilia Encode.mlx file
3. Run the MyJANUSDemo.mlx file
4. Run the Venilia Decode.mlx file
5. Open the message.txt file under the build directory
